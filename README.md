一个Windows端的简易工具，具体参数如下。希望师傅们喜欢这个项目
 -ip: 显示所有IP配置信息。
 -a: 显示系统信息。
 -w: 显示所有运行中的进程。
 -b: 显示本地机器上的所有服务。
 -c: 显示计划任务的详细信息。
 -d: 显示工作站统计信息。
 -e: 显示本地机器上的用户账户信息。
 -f: 显示管理员组的成员。
 -g: 显示所有监听端口和相关进程信息。
 -h: 显示此帮助消息。
 -i: 显示本地机器上的所有共享资源。
 -j: 使用wmic显示所有共享资源的名称、路径和状态。
 -k: 使用route print显示路由表。
 -l: 使用arp -a显示所有ARP缓存条目。
 -m: 使用whoami /all显示当前用户的详细信息。
 -n: 显示工作站配置信息。
 -cc: 从源路径复制文件到目标路径。
 -cv: 从源路径移动文件到目标路径。
 -x: 在指定路径创建新文件并写入内容。
 -v: 查看指定目录的所有文件。
 -o: 使用fsutil fsinfo drives显示所有可用驱动器。
 -all: 执行所有命令。
