一个Windows端的简易工具，具体参数如下（汉化会出现乱码，工具处于测试）
# -ip: 显示所有IP信息。
# -s "搜索字符串": 在文件名或内容中搜索指定的字符串。
# -u URL: 在默认网页浏览器中打开指定的URL。
# -p URL: Ping指定的URL。
# -w: 列出所有运行的进程。
# -kill PID: 杀死指定PID的进程。
# -h: 显示帮助信息。
# -c URL [保存路径]: 从指定的URL下载文件到给定的保存路径。
# -cc 源路径 目标路径: 复制文件从源路径到目标路径。
# -cv 源路径 目标路径: 移动文件从源路径到目标路径。
# -v 目录路径: 查看指定目录的位置。
# -k 文件路径 [内容]: 创建一个新文件，指定文件路径并写入内容。
